@GeneratedValue 注解作用是指定该字段值（这里是主键值）的生成策略，strategy = GenerationType.IDENTITY 表示使用自增的方式生成主键值，即使数据库中的主键字段已经定义了自增主键，在 Java 代码中仍然推荐使用该注解，一方面是可以增强代码的可读性，另一方面是 JPA 和 Hibernate 需要这样的元数据来正确地映射实体类到数据库表，最后，也可以增强代码的可移植性。
