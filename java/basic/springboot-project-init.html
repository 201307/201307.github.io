Spring Boot项目初始化

  • JAVA

新建项目

在 IDEA 中进行如下操作：

File – New – Project，点击后会弹出对话框。

在对话框的左侧点击 Spring Boot，则会出现 Spring Boot 项目初始化的配置项目，如图：

Spring Boot 官方配置项目中可能已不支持低版本的 Java 选项，此时需要编辑对话框顶部的 Server URL 项，点击右侧的设置图标后，填入阿里的服务配置地址：https://start.aliyun.com 点击 OK 后即可支持低版本 Java 配置。

根据实际情况依次填写所有配置项，然后点击 Next 按钮。

新出现依赖配置的对话框，用于选择 Spring Boot 版本及项目内所需要的 依赖项（也可后续在项目内手动修改 pom 文件进行配置），根据实际情况填写后点击 Create 按钮即可完成项目的创建。

完善配置

运行配置

点击 IDEA 中进行如下操作：

Current File – Edit Configurations… ，点击后会弹出对话框。

点击对话框左上角的 + 图标，然后再点击 Application 选项，则会出现运行配置项，根据实际情况填写完成后点击 OK 按钮即可。

项目启动

由于此前配置依赖时直接勾选了 Web 依赖，所以项目中已经初始化好了基础的 Web 配置和示例代码，项目可以直接启动。

点击 IDEA 右上角的 Run 图标 或者 Debug 图标即可启动项目。

根据示例代码中的访问地址，在浏览器中进行访问。
